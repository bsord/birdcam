## Launch & Usage

Install prerequisites:

`pip3 install -r requirements.txt`


Basic start-up:

`python3 main.py`


Advanced start-up:

`python3 main.py --port 8000 --debug --slack-token=[SLACKTOKEN] --slack-channel='#birdcam' --detection-delay=10 `

View Live feed:

Open browser to http://localhost:8000
